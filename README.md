# 🌊 MediaWave – Digital Media Website

A modern and responsive digital media website built using HTML and CSS.  
This project focuses on clean UI design, professional background sections, and structured content layout.

---

## 🚀 Features

- 🎬 Hero section with professional background image
- 🔥 Trending media section with styled background
- 📖 About section with continuous background design
- 🌙 Footer section matching overall theme
- 📱 Fully responsive layout
- 🎨 Bright and modern UI design
- 🧭 Clean navigation bar

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- Responsive Web Design Principles
